# Simple-IG
Just a Simple Incremental Game, Tap the button to generate points, upgrade your button, and make your points larger
# How to Play
Download the game and open the "index.html" file
# Features
- Restores your data using `LocalStorage` so it will never leave your device unless deleted
- Upgradeable mechanics
- Prestige mechanics
# Dependencies

- [decimal.js](https://github.com/MikeMcl/decimal.js)
